Project 2

Tangible generative music system

For my final year project of university, I created a tangible generating music system taking advantage of Human Computer Interaction techniques and Tangible User Interfaces. The goal of this project was to explore the effects of introducing tangibles to music creation for both layman and virtuoso. In this case the tangibles used were small digital devices called M5 Stacks with sensors to detect manipulation. This project was implemented with both a Tangible and Graphical User Interface. The GUI was developed using Java in the Processing 3 environment where as the TUI using C in the Arduino environment.
